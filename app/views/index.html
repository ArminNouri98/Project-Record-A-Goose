'Record a goose sighting' accessibility exercise instructions

'Record a goose sighting' is a made-up government service, filled with accessibility issues. Your task is to find them all! If you'd like to know more about the background, visit the blog post about this, 'Training people to do accessibility reviews'.

To identify the issues, you'll want to use keyboard testing, and WAVE. More details are in a worksheet you'll need to accompany this exercise.

There are 6 pages in total:

  • Start page
  • Do you like geese?
  • What type of goose did you see?
  • When did you see the goose?
  • Check your answers
  • Confirmation page

Accessibility issues start on the next page. Good luck!
